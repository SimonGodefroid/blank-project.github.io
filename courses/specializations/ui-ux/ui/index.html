Couleurs et palettesarrow_right

Les objectifs de ce cours sont :

  • Connaître les notions fondamentales de la théorie des couleurs
  • Connaître les techniques de sélection d'un ensemble de couleurs
  • Comprendre le modèle HSB
  • Utiliser les composantes du modèle HSB pour obtenir des variations de couleurs
  • Savoir créer une palette de couleur pour le web design
Typographiearrow_right

Les objectifs de ce cours sont :

  • Connaitre les propriétés d'une police de caractères
  • Connaitre les types de polices et la typographie associée
  • Être capable de trouver des combinaisons de polices harmonieuses
  • Connaitre et savoir utiliser les propriétés de lisibilité
  • Savoir mettre en valeur un texte
Icônesarrow_right

Les objectifs de ce cours sont :

  • Connaitre la classification et les propriétés des icônes
  • Savoir identifier les types d'icônes selon la fonction
  • Savoir identifier les types d'icônes selon le style visuel
  • Pouvoir mettre en oeuvre les bonnes pratiques liées à l'utilisation des icônes sur le web
UI : Maquette, Moodboard, Figmaarrow_right

Les objectifs de ce cours sont :

  • Savoir élaborer un moodboard
  • Savoir créer des wireframes
  • Savoir décomposer des wireframes en composants graphique et les mettre en forme sur Figma
  • Savoir créer une maquette graphique complète (moodboard, composants, couleurs, polices, wireframes)
