Généralités

Utiliser le Terminal - Part 1arrow_right

Les objectifs de ce cours sont :

  • Comprendre le principe d'un terminal.
  • Connaître les différents terminaux.
  • Savoir exécuter des commandes basiques.
  • Savoir exécuter des commandes avec des arguments.
  • Connaître le principe du "chemin d'accès" ou "path".
Utiliser le Terminal - Part 2arrow_right

Les objectifs de ce cours sont :

  • Connaître le principe de scripts shell.
  • Savoir créer des scripts shell.
  • Savoir chaîner des commandes.
  • Savoir gérer les sorties d'une commande.
Git & Githubarrow_right

Les objectifs de ce cours sont :

  • Connaître les principes de Git.
  • Connaître les principes de Github.
  • Savoir utiliser Git à l'aide d'un terminal.
  • Savoir utiliser le système de Fork / Pull Request / Issues de Github.
Introduction à la programmation - Part 1arrow_right

Les objectifs de ce cours sont :

  • Comprendre le principe de Logo.
  • Savoir utiliser les principales commandes Logo.
Introduction à la programmation - Part 2arrow_right

Les objectifs de ce cours sont :

  • Comprendre les principes de base de la programmation (tests, boucles, types de valeur, fonctions, variables).
  • Savoir appliquer chacun d'entre eux dans un exemple concret et l'expliquer.
Le déploiementarrow_right

Les objectifs de ce cours sont :

  • Comprendre le principe de packaging.
  • Savoir utiliser SSH pour se connecter à distance à un terminal.
  • Connaitre les différentes méthodes de déploiement.
