Bases de données

Introductionarrow_right

Les objectifs de ce cours sont :

  • Savoir installer MongoDB.
  • Comprendre ce que sont les bases de données.
  • Connaître la différence entre une base de données documents et relationnelle.
  • Savoir installer MongoDB.
  • Savoir intéragir avec le shell MongoDB pour des commandes basiques.
Manipuler les donnéesarrow_right

Les objectifs de ce cours sont :

  • Savoir manipuler les données avec le shell MongoDB (Insertion, Récupération, Mise à jour, Suppression).
  • Connaitre les différents opérateurs de requêtes (Comparaison, Logiques, Elements, Evaluation,...).
NodeJS + MongoDBarrow_right

Les objectifs de ce cours sont :

  • Comprendre le principe d'un driver.
  • Savoir installer le driver MongoDB pour Node.js.
  • Connaître les deux modes du driver MongoDB pour Node.js.
  • Savoir initialiser un client MongoDB au sein d'une application Node.js.
  • Savoir utiliser un client MongoDB pour Insérer, Lire, Modifier et Supprimer des données (CRUD).
Mongoosearrow_right

Les objectifs de ce cours sont :

  • Comprendre le principe des ORM.
  • Savoir gérer les connexion à une base de données MongoDB avec mongoose.
  • Connaître les différents principes de mongoose (schema, model, types de valeurs, etc.).
  • Savoir créer des Models et Schémas mongoose.
  • Savoir utiliser mongoose pour gérer les données d'une base MongoDB.
  • Savoir utiliser mongoose avec les promises et async await (ES6+).
